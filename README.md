# Magnus

Magnus is a lightweight desktop dashboard for **post-trade analysis of pair trades activity**.

It provides interactive views to analyze **PnL, trades, pairings (L1/L2/L3), counterparties and underlyings**, with fast filtering and clear visual KPIs.

### Features
- Pairing analysis (L1 / L2 / L3)
- Counterparty & underlying PnL
- Cumulative and daily views
- Interactive tables and charts

### Tech
Python · Tkinter · Matplotlib · Pandas

> Internal analytics tool focused on clarity, performance and extensibility.
